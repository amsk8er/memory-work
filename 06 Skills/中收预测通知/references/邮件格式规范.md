# 邮件格式规范

## 字体规范

| 元素 | 字体 | 字号 | 样式 |
|---|---|---|---|
| 大标题（通知标题） | 宋体, SimSun, serif | 20pt | 居中加粗 |
| 小标题（一、二、三、四） | 黑体, SimHei, sans-serif | 14pt | 加粗 |
| 正文 | 仿宋, FangSong, serif | 14pt | 首行缩进2em |
| 重点强调（日期、加报提醒） | 黑体, SimHei, sans-serif | 14pt | 加粗 |
| 紧急内容（如18时加报） | 黑体, SimHei, sans-serif | 14pt | 加粗 + 红色(#CC0000) |

## 通用样式

- 正文行高: 1.8
- 页面边距: margin 40px 60px
- 文字颜色: #000000
- 落款: 右对齐, margin-right 60px

## 日历表格样式（仅市分行通知）

| 元素 | 样式 |
|---|---|
| 表头行 | 背景 #335294, 白色字, 黑体 |
| 普通单元格 | border 1px solid #999, padding 8px |
| 报送日期 | 背景 #FFF3CD（淡黄）, 加粗, 数字后加 ✅ |
| 特殊日期（末日加报） | 背景 #FFD7D7（淡红）, 加粗 ✅ + 小字标注"18时加报"(9pt, #CC0000) |
| 周末日期 | 灰色字体 #999 |
| 表��字体 | 仿宋 12pt, 居中 |

## 收件人规则

- **不使用公共信箱**（如 jszhpj_cwhj_js@mails.icbc 等）
- To 和 Cc 中的姓名用双引号包裹
- 抄送固定为: 陈良元、徐捷（省分行财务会计部）
- 收件人通讯录位置: `03 Projects/中收预测/收件人通讯录.md`

## 附件说明

- 市分行通知: 附报送表样（邮件中注明"表样见附件"）
- 部门通知: 附科目预测模板（正文中以蓝色下划线样式显示文件名）
  - 附件占位样式: `color: #0563C1; text-decoration: underline`
