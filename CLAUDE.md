# Memory Work - Claude 初始化指令

> 这是一个全新的 Memory Work 项目。当你在这个仓库中被启动时，请遵循以下初始化流程。

---

## 首次启动检测

**每次会话开始时，检查以下条件判断是否需要初始化**：

1. 检查 `USER.md` 是否还是模板状态（包含 `[Your Name]` 或 `[你的名字]`）
2. 检查 `00 Focus Zone/_this_week.md` 或 `00 专注区/_本周.md` 是否存在且有实际内容

**如果是首次使用（USER.md 未配置）**：
→ 执行下方的「首次启动引导」

**如果已配置**：
→ 按 `AGENTS.md` 的正常启动流程执行

---

## 首次启动引导

### Step 1: 语言选择

向用户询问：

```
你好！欢迎使用 Memory Work 🎉

这是一个让 AI 拥有持久记忆的知识管理系统。

首先，请选择你的偏好语言：
1. 中文
2. English

输入 1 或 2：
```

### Step 2: 初始化文件

**如果用户选择「中文」**：

1. 将 `templates/zh-CN/` 下的所有文件覆盖到根目录：
   - `templates/zh-CN/AGENTS.md` → 覆盖根目录 `AGENTS.md`
   - `templates/zh-CN/SOUL.md` → 覆盖根目录 `SOUL.md`
   - `templates/zh-CN/USER.md` → 覆盖根目录 `USER.md`
   - `templates/zh-CN/MEMORY.md` → 覆盖根目录 `MEMORY.md`
   - `templates/zh-CN/PROCEDURES.md` → 覆盖根目录 `PROCEDURES.md`

2. 复制中文专注区并删除英文专注区：
   - `templates/zh-CN/00 专注区/` → 复制为 `00 专注区/`
   - **删除** `00 Focus Zone/` 目录及其所有内容

3. 创建中文目录结构：
   ```
   00 专注区/
   01 素材库/
   02 工具库/
   06 技能库/
   ```

**如果用户选择「English」**：

1. **删除** `templates/zh-CN/00 专注区/` 目录（避免混淆）

2. 保持当前英文文件不变，创建英文目录结构：
   ```
   00 Focus Zone/
   01 Materials/
   02 Tools/
   06 Skills/
   ```

### Step 3: 用户画像配置

**不要用问卷式的方式收集信息。** 像认识一个新朋友一样聊天，但要高效——一轮可以问 2-3 个相关问题：

```
好，中文。

那我先认识一下你——你现在主要在忙什么？做什么工作的？怎么称呼你？
```

用户回应后，**自然地确认和补充**，比如：

- 「老王，产品经理，AI 教育方向。这个产品是 to B 还是 to C 的？你在里面主要负责什么？」
- 「创业者啊，现在是什么阶段？有团队还是单干？」

**目标不是填满 USER.md 的每个字段**，而是快速建立基本认知：
- 怎么称呼
- 现在的工作和角色
- 当前最关注的事

了解基本情况后，**主动引导用户提供更多 context**：

```
行，大概了解了。我先记一下——

[简单总结写入 USER.md]

---

对了，如果你手边有一些能帮我更快了解你的资料，可以随时发给我：

- 你的个人介绍、简历、或者之前写过的「关于我」
- 最近在做的项目文档、PRD、或者会议纪要
- 你关注的领域的文章、笔记
- 甚至工作截图、聊天记录截图都行

不用一次给全，想到什么发什么。这些能帮我更快进入状态，之后聊起来也更顺。
```

### Step 4: 工作区创建与本周启动

```
核心文件建好了：
- USER.md — 你的档案
- MEMORY.md — 长期记忆
- _本周.md — 这周的工作台

---

今天 [星期几]，聊聊这周的情况？

有什么在忙的、悬着的、或者 deadline 在追你的？如果有相关的文档、邮件截图、会议纪要什么的，也可以一起发给我，我能更快帮你理清楚。
```

**处理用户输入时**：
- 先接住他说的，理解这周的处境和轻重缓急
- 如果用户只是口述，可以追问：「培训那个有 PPT 或者大纲了吗？发给我看看？」
- 如果用户发了文档/截图，先消化内容，再结合他说的整理

**整理完任务后**，确认主线和支线，然后自然地问：
- 「有确定时间的事情，要我生成日历文件吗？」

### Step 5: Obsidian 引导（如果用户问）

如果用户询问如何在 Obsidian 中打开，回答：

```
在 Obsidian 中打开这个文件夹：

1. 打开 Obsidian
2. 点击左下角的「打开其他仓库」（vault 图标）
3. 选择「打开文件夹作为仓库」
4. 导航到这个路径：

   [当前工作目录的绝对路径]

或者直接在 Finder 里 Cmd + Shift + G，粘贴上面的路径，就能看到这个文件夹的位置。
```

---

## 初始化完成后

完成初始化后，后续会话按正常流程运行：

1. 读取 `SOUL.md` → 了解你是谁
2. 读取 `USER.md` → 了解你在帮助谁
3. 读取 `_本周.md` / `_this_week.md` → 获取当前工作上下文
4. 按需加载 `MEMORY.md` 和 `PROCEDURES.md`
5. 按需搜索全库
