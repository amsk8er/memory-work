## 领导额外要求

### 核心主线
金融强国建设 → 服务实体经济 + 防控风险 → 高质量发展 → 正确政绩观（一脉相承）

### 发言结构（领导梳理版）

**Part 1：围绕学习材料二，谈高质量发展学习体会**

1. **正确的政绩观**——什么是正确的政绩观？
   - 不拼规模、不比增速、不分排名
   - 践行科学发展观，追求高质量发展

2. **如何践行正确的业绩观？**（年度工作会议讲话 / 高质量发展规划）
   - 做好宣导，对下提倡实事求是、做好调研
   - 围绕6项具体工作举措：
     - 价值创造力提升
     - 市场竞争力提升
     - 可持续发展能力提升
     - 客户基础夯实
     - 全方位服务实体经济
     - 优化结构、掌握竞争主动

3. **怎么做好监督**（业务合规）
   - 参考：《关于印发〈资金使用效率提升实施细则（2025年版）〉的通知》
   - 贷款：实质性贷款增长 + 贷款均衡度
   - 存款：稳定性、日均存款、回撤率
   - 中收/收费合规：去年退费4亿元，今年已退还部分
   - 加强过程管理

4. **怎么做好考核评价**
   - 参考：《资金使用效率提升实施细则（2025年版）》中存贷款业务考核
   - 参考：《境内分行KPI考核和等级行评定办法（2026年版）》第十五条 激励约束
   - KPI办法原文引用：
     > "加强对不当经营占用科目、不当用益和定期考核数据的检查，对异常情况进行监测、识别和处罚。对弄虚作假或财务数据失真的，贷款评级将作为考核评价和激励约束的依据，并加倍惩戒。"
