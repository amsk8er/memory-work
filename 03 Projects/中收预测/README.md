# 中间业务收入预测项目

## 项目概述

根据总行要求，每季度末月需组织各市分行及省分行专业部门报送中间业务收入预测数据。省分行财务会计部负责汇总上报总行。

## 工作流程

### 1. 接收总行通知
- 总行财务会计部中间业务处下发预测通知
- 明确报送内容、方式、时间节点
- 总行报送渠道：FRMS系统，报表分类"中间业务处"，报表"中间业务收入及支出预测表（FCFORCAST）"

### 2. 对照省行财务预测通知
- 省行财务会计部会同步下发财务收支预测通知（41表）
- 中收预测报送时间节点需与省行财务预测通知**保持一致**
- 如总行有额外时间要求（如首次报送提前），需增加报送节点

### 3. 起草通知（两封邮件）

每个季度需要发送**两封通知**：

#### 通知A：给市分行
- 收件人：各市分行财会部联系人（不含省行部门人员）
- 核心要素：
  - **预测内容**：手佣毛收入及分项、手佣支出、净收入（按业务条线明细）；科目包括5310、522012
  - **报送方式**：邮件报送，附表样
  - **报送时间**：与省行财务预测一致 + 总行额外节点；最后一天18时加报
  - **注意事项**：口径含分润和考核还原（含工银理财），信用卡总行分润不再单独报送；净收入同比下降需书面说明
- 正文含日历表格，报送日期标 ✅
- C列填写规则：首次报送填上月末实际数，后续填上日实绩，末日18时填17时预查询结果

#### 通知B：给省分行专业部门
- 收件人：省分行各专业部门（不含财务会计部本部）
- 核心要素：
  - **报送内容**：分科目收入/支出预测 + 季末下划收入预测，共两张表
  - **报送时间**：通常2次，首次与末次对齐总行关键节点
  - **填报提示**：含总行下划/分润、省行划转、工银理财模拟考核；支出涉及银行卡/个金/结现等；科目分工按上年分润规则
  - **预测表样**：附同期、上月科目收入及预测累计数
- 附件：省分行专业部门科目预测模板

### 4. 邮件发送（两封通知共同规则）
- **格式要求**：大标题宋体、小标题黑体、正文仿宋、重点内容黑体加粗
- **抄送**：省分行财务会计部 陈良元、徐捷
- **不使用公共信箱**
- 收件人详见 `03 Projects/中收预测/收件人通讯录.md`

### 5. 日常跟踪
- 按时间节点催收各行报送
- 汇总数据上报总行（通过FRMS系统）
- 关注净收入同比下降的行，收集原因说明

## 时间线模板（以一季度为例）

| 阶段 | 时间 | 动作 |
|---|---|---|
| 收到总行通知 | 2月底 | 分析总行要求 |
| 对照省行通知 | 2月底 | 确认时间节点 |
| 起草并发送通知 | 2月28日 | 两封邮件：市分行通知 + 部门通知 |
| 部门首次报送 | 3月初 | 收集部门科目预测 |
| 市分行首次报送 | 3月初 | 收集市分行首轮预测 |
| 持续报送 | 3月中下旬 | 按节点跟踪（市分行多次、部门第二次） |
| 末日加报 | 3月31日18时 | 最终预测 |
